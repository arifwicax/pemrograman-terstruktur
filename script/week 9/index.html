🚀 Week 9 - Array dalam JavaScript

Belajar konsep array, method-method array, dan implementasi praktis

📋 Petunjuk Penggunaan:

  1. Buka Developer Console (F12 atau Ctrl+Shift+I)
  2. Klik tombol "Jalankan" untuk menjalankan setiap script
  3. Lihat output di console untuk memahami setiap konsep
  4. Uncomment script di bawah untuk menjalankan otomatis

💡 Tip: Jalankan script satu per satu untuk pemahaman yang lebih baik!

1️⃣ Array Basics

Mempelajari konsep dasar array, index, property length, dan karakteristik array JavaScript

2️⃣ Array Creation Methods

Berbagai cara membuat array: literal, constructor, Array.of(), Array.from(), dan spread operator

3️⃣ Array Access & Manipulation

Mengakses, mengubah, menambah, dan menghapus elemen array dengan berbagai method

4️⃣ Array Methods Dasar

Method-method dasar: push, pop, shift, unshift, indexOf, includes, join, concat, slice

5️⃣ Array Iteration

Berbagai cara melakukan perulangan: for, for...of, forEach, map, filter, reduce

6️⃣ Array Methods Advanced

Method advanced: sort, flat, flatMap, splice, fill, copyWithin, chaining methods

7️⃣ Multidimensional Arrays

Array 2D, 3D, dan aplikasi praktis seperti matrix, game board, dan tabel data

8️⃣ Comprehensive Examples

Implementasi nyata: sistem mahasiswa, inventory toko, game battleship, dan algoritma

📝 Catatan Penting:

  • Array dimulai dari index 0, bukan 1
  • Array di JavaScript dynamic - ukuran bisa berubah
  • Array bisa menyimpan berbagai tipe data
  • Method array ada yang mengubah array asli dan ada yang return array baru
  • Gunakan method yang tepat untuk setiap kebutuhan

🔄 Method yang Mengubah Array Asli:

push, pop, shift, unshift, splice, sort, reverse, fill

✨ Method yang Return Array Baru:

map, filter, slice, concat, flat, flatMap
