Week 4 - Types, Values, dan Variables pada Javascript

📚 Cara Menggunakan:

  1. Buka Developer Tools (F12 atau Ctrl+Shift+I)
  2. Pilih tab Console
  3. Uncomment salah satu baris <script> di bawah
  4. Refresh halaman (F5) untuk menjalankan script
  5. Lihat output di Console
⚠️ Penting: Hanya aktifkan satu script dalam satu waktu untuk menghindari konflik output di console.

📋 Daftar Script:

1. Primitive Types - Menjelaskan tipe data dasar: Number, String, Boolean, null, undefined
2. Number Operations - Operasi matematika, Math object, increment/decrement
3. String Manipulation - Methods string, template literals, manipulasi teks
4. Variables & Constants - let vs const, scope, hoisting, naming conventions
5. Type Conversion - Konversi tipe data implisit dan eksplisit
6. Objects - Object literals, methods, nested objects, destructuring
7. Comprehensive Example - Program lengkap sistem akademik yang menggabungkan semua konsep
