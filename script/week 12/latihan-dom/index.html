Latihan DOM JavaScript - Pembelajaran Terstruktur

Selamat Datang

Folder ini berisi 36 latihan terstruktur untuk mempelajari DOM (Document Object Model) JavaScript dari dasar hingga mahir.

Cara belajar: Ikuti urutan dari nomor 1-36, buka setiap file di browser dan lihat Console (F12) untuk penjelasan detail.

Bagian 1: Dasar Window & Document (1-4)

  • 01 - Objek Window
    Memahami objek window sebagai objek global browser
  • 02 - Objek Document
    Mengenal objek document yang merepresentasikan halaman HTML
  • 03 - Properti Document
    Mengakses properti document (title, head, body, images)
  • 04 - Method Document
    Menggunakan method document untuk mengakses element

Bagian 2: Selector Element (5-10)

  • 05 - Selector ID
    Memilih element dengan selector ID (#)
  • 06 - Selector Class
    Memilih element dengan selector class (.)
  • 07 - Selector Type (Tag)
    Memilih element berdasarkan nama tag HTML
  • 08 - Selector Universal
    Memilih semua element dengan selector universal (*)
  • 09 - Selector Attribute
    Memilih element berdasarkan keberadaan attribute
  • 10 - Selector Attribute dengan Operator
    Selector attribute dengan operator (^=, $=, *=)

Bagian 3: Element & Properti (11-14)

  • 11 - Objek Element
    Memahami objek element dalam DOM
  • 12 - Properti Element
    Mengakses properti element (id, className, tagName, innerHTML)
  • 13 - Method Element
    Menggunakan method element untuk manipulasi (remove)
  • 14 - Membuat Element
    Membuat element baru dengan createElement dan appendChild

Bagian 4: Node & NodeList (15-21)

  • 15 - Objek Node
    Memahami node sebagai objek dasar dalam DOM tree
  • 16 - Properti Node
    Navigasi antar node (childNodes, previousSibling, nextSibling)
  • 17 - Method Node
    Memanipulasi node dengan removeChild
  • 18 - Tipe Node
    Mengenal berbagai tipe node (element, text, comment)
  • 19 - Objek NodeList
    Memahami NodeList sebagai koleksi nodes
  • 20 - NodeList Static
    NodeList static dari querySelectorAll (tidak live)
  • 21 - NodeList Live
    NodeList live dari childNodes (otomatis update)

Bagian 5: Attribute & Style (22-26)

  • 22 - Objek Attribute
    Mengakses attribute element dengan getAttribute
  • 23 - Method Attribute
    Memanipulasi attribute dengan setAttribute
  • 24 - Membuat Attribute
    Menambahkan attribute baru ke element
  • 25 - Named Node Map
    Mengakses semua attribute melalui NamedNodeMap
  • 26 - Properti Style
    Mengubah style CSS element secara dinamis

Bagian 6: Text & HTML Content (27-29)

  • 27 - Inner Text
    Perbedaan textContent, innerText, dan innerHTML
  • 28 - Inner HTML
    Mengatur konten HTML dengan innerHTML
  • 29 - Membuat Text Node
    Membuat node teks dengan createTextNode

Bagian 7: Event Handling (30-32)

  • 30 - Objek Event
    Memahami objek event dan propertinya
  • 31 - Event Target
    Menggunakan addEventListener untuk event handling
  • 32 - Event Global
    Event handler dengan onclick (tanpa addEventListener)

Bagian 8: HTML Element Spesifik (33-36)

  • 33 - HTML Element
    Properti spesifik HTML element (contoh: anchor)
  • 34 - HTML Form (Method 1)
    Menangani form dengan querySelector
  • 35 - HTML Form (Method 2)
    Menangani form dengan document.forms
  • 36 - HTML Table
    Mengakses bagian-bagian table (caption, thead, tbody, tfoot)

Tips Belajar

  • Ikuti urutan dari file 01 hingga 36 secara berurutan
  • Buka Console (tekan F12) di browser untuk melihat penjelasan detail
  • Ubah kode dan lihat hasilnya untuk memahami lebih dalam
  • Coba implementasikan konsep yang dipelajari ke project sendiri
  • Jangan ragu untuk kembali ke materi sebelumnya jika perlu
