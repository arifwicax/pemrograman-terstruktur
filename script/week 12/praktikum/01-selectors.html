Praktikum 1: Element Selectors

Belajar cara memilih elemen HTML menggunakan berbagai metode selector

1. getElementById()

Memilih elemen berdasarkan ID (harus unik)

Special Box - ID: special-box
const element = document.getElementById('special-box'); element.style.background = 'lightblue';

2. getElementsByClassName()

Memilih semua elemen dengan class tertentu (HTMLCollection)

Apple
Banana
Carrot
Cherry
const fruits = document.getElementsByClassName('fruit'); // fruits = HTMLCollection [div.fruit, div.fruit, div.fruit] for(let fruit of fruits) { fruit.classList.add('highlight'); }

3. getElementsByTagName()

Memilih semua elemen dengan tag tertentu

Paragraf 1

Div 1

Paragraf 2

Span 1

Paragraf 3

const paragraphs = document.getElementsByTagName('p'); for(let p of paragraphs) { p.classList.add('highlight'); }

4. querySelector()

Memilih SATU elemen pertama yang cocok dengan CSS selector

Box 1 (priority: high)
Box 2 (priority: low)
Box 3 (priority: high)
// Pilih box pertama dengan priority="high" const highPriorityBox = document.querySelector('.box[data-priority="high"]'); highPriorityBox.classList.add('highlight');

5. querySelectorAll()

Memilih SEMUA elemen yang cocok dengan CSS selector (NodeList)

Item 1 (active)
Item 2 (inactive)
Item 3 (active)
Item 4 (active)
// Pilih SEMUA item dengan status="active" const activeItems = document.querySelectorAll('.item[data-status="active"]'); activeItems.forEach(item => { item.classList.add('highlight'); });

6. Perbandingan & Best Practice

Tips Memilih Selector:
• getElementById() - Paling cepat, gunakan untuk ID unik
• querySelector() - Paling fleksibel, bisa pakai CSS selector apapun
• querySelectorAll() - Untuk pilih banyak elemen dengan pola CSS
• getElementsByClassName() - Live collection (auto update)
• getElementsByTagName() - Live collection (auto update)
Cara Praktik:
1. Klik tombol pada setiap section untuk melihat hasilnya
2. Perhatikan elemen mana yang ter-highlight (warna kuning)
3. Buka DevTools Console (F12) untuk melihat log detail
4. Klik "Reset" untuk menghapus semua highlight
5. Coba modifikasi kode di DevTools Console untuk eksperimen
