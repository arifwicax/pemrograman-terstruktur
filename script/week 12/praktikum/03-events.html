6. Event Object Properties

Setiap event handler menerima object event dengan informasi detail

Event Object Properties:
• event.target - Element yang memicu event
• event.type - Jenis event (click, input, dll)
• event.clientX/Y - Posisi mouse
• event.key - Tombol keyboard yang ditekan
• event.preventDefault() - Mencegah aksi default
• event.stopPropagation() - Menghentikan event bubbling
Tips Praktik:
• Buka DevTools Console (F12) untuk melihat log detail
• Semua event log ditampilkan di box hitam di setiap section
• addEventListener() lebih baik daripada onclick attribute
• Event object berisi banyak informasi berguna
• Gunakan preventDefault() untuk mencegah aksi default (misal: form submit)
