Praktikum 4: Create & Remove Elements

Belajar membuat, menambahkan, dan menghapus elemen DOM secara dinamis

1. createElement() & appendChild()

Membuat elemen baru dan menambahkan ke DOM

// Buat element baru const div = document.createElement('div'); div.textContent = 'Item Baru'; div.className = 'item'; // Tambahkan ke parent container.appendChild(div);

2. remove() & removeChild()

Menghapus elemen dari DOM

// Method 1: remove() - Modern way element.remove(); // Method 2: removeChild() - Classic way parent.removeChild(element);

3. insertBefore()

Menambahkan elemen di posisi tertentu

// Insert sebelum element tertentu const newElement = document.createElement('div'); parent.insertBefore(newElement, referenceElement); // Insert di awal (sebelum child pertama) parent.insertBefore(newElement, parent.firstChild);

4. cloneNode()

Menduplikasi elemen yang sudah ada

Template Card

Ini adalah template card yang bisa di-clone

// Shallow clone (tanpa children) const shallowClone = element.cloneNode(false); // Deep clone (dengan semua children) const deepClone = element.cloneNode(true);

5. innerHTML vs createElement

Perbandingan dua cara membuat konten HTML

Menggunakan innerHTML

Menggunakan createElement

Kapan menggunakan mana?
• innerHTML: Cepat untuk HTML statis, tapi bisa XSS vulnerable
• createElement: Lebih aman, lebih kontrol, recommended untuk dynamic content
