Praktikum 2: Manipulasi Konten & Properties

Belajar mengubah konten, attribute, style, dan class pada elemen HTML

1. textContent - Mengubah Teks

Property untuk mengambil atau mengubah teks mentah (tanpa HTML)

Ini adalah teks original
const element = document.getElementById('text-demo'); element.textContent = 'Teks telah diubah!';

2. innerHTML - Mengubah HTML

Property untuk mengambil atau mengubah konten HTML (termasuk tag)

Konten HTML original

const element = document.getElementById('html-demo'); element.innerHTML = '<strong>Teks Bold</strong> dan <em>Teks Miring</em>';

3. Attribute - Mengubah Attribute Element

Menggunakan getAttribute(), setAttribute(), dan removeAttribute()

Element dengan attributes
Current Attributes:
element.setAttribute('data-status', 'active'); element.setAttribute('data-priority', 'high'); const status = element.getAttribute('data-status');

4. Style - Mengubah CSS Inline

Mengubah style langsung via JavaScript (inline CSS)

Style Box
element.style.backgroundColor = '#e74c3c'; element.style.color = 'white'; element.style.padding = '30px'; element.style.transform = 'rotate(5deg)';

5. classList - Manipulasi CSS Class

Menambah, menghapus, dan toggle class CSS

Element dengan classList
Current Classes:
element.classList.add('active'); element.classList.remove('active'); element.classList.toggle('featured'); element.classList.contains('active'); // true/false

6. dataset - Data Attributes

Akses data-* attributes dengan mudah

Element dengan data attributes
Dataset:
element.dataset.userId = '456'; element.dataset.userName = 'Jane Smith'; console.log(element.dataset.userRole); // 'admin'
Tips Praktik:
• Buka DevTools Console (F12) untuk melihat log detail
• Coba modifikasi nilai di input field sebelum klik tombol
• Perhatikan perbedaan textContent vs innerHTML
• classList lebih baik daripada className untuk manipulasi class
• Dataset otomatis convert camelCase (data-user-name → dataset.userName)
