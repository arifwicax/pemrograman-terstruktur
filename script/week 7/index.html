Week 7 - Object dalam JavaScript

📚 Cara Menggunakan:

  1. Buka Developer Tools (F12 atau Ctrl+Shift+I)
  2. Pilih tab Console
  3. Uncomment salah satu baris <script> di bawah
  4. Refresh halaman (F5) untuk menjalankan script
  5. Lihat output di Console
⚠️ Penting: Hanya aktifkan satu script dalam satu waktu untuk menghindari konflik output di console.

📋 Daftar Script:

1. Object Basics - Konsep object, key-value, karakteristik, nested object, referensi
2. Object Creation Methods - Object literal, constructor, new Object(), Object.create(), ES6 class
3. Property Operations - Akses (dot/bracket notation), ubah, hapus, cek property
4. Object Methods & Functions - Method, this keyword, getter/setter, method chaining
5. Object Iteration - for...in, Object.keys(), Object.values(), Object.entries()
6. Prototype & Inheritance - Prototype chain, constructor inheritance, ES6 class extends, mixins
7. Comprehensive Example - Program lengkap Sistem Perpustakaan yang menggabungkan semua konsep

📖 Materi yang Dipelajari:

  • Object Basics: Konsep object, key-value pairs, nested object, referensi vs copy
  • Object Creation: Literal, constructor function, Object.create(), ES6 class
  • Property Operations: Dot/bracket notation, CRUD property, Object.assign(), spread
  • Methods: Function dalam object, this keyword, getter/setter
  • Iteration: for...in, Object.keys/values/entries, kombinasi array methods
  • Prototype: Prototype chain, inheritance, extends, super, mixins
  • Modern ES6: Shorthand, spread operator, destructuring, computed property
  • Serialisasi: JSON.stringify() dan JSON.parse()
