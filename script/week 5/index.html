🎯 Konsep Utama Week 5:

🔢 Arithmetic Operators

  • +, -, *, /, % (Basic)
  • ** (Exponentiation)
  • ++, -- (Increment/Decrement)
  • Unary +, - operators

⚖️ Comparison Operators

  • ==, === (Equality)
  • !=, !== (Inequality)
  • >, <, >=, <= (Relational)
  • Type coercion vs strict

🔗 Logical Operators

  • && (AND operator)
  • || (OR operator)
  • ! (NOT operator)
  • Short-circuit evaluation

📝 Assignment Operators

  • = (Basic assignment)
  • +=, -=, *=, /= (Compound)
  • Destructuring assignment
  • Multiple assignments

🔧 Special Operators

  • typeof (Type checking)
  • delete (Property removal)
  • ?: (Ternary operator)
  • instanceof, in operators

🚀 Modern Operators

  • ?? (Nullish coalescing)
  • ?. (Optional chaining)
  • ... (Spread operator)
  • Comma operator
