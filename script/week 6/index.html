📝 Week 6 - Statements dalam JavaScript

Deskripsi Materi:
Pada week ini kita mempelajari Statements (Pernyataan) dalam JavaScript. Statement adalah perintah dalam program yang dijalankan oleh JavaScript. Program JavaScript pada dasarnya adalah sekumpulan statement yang dijalankan secara berurutan.

🎯 Tujuan Pembelajaran

  • Memahami pengertian dan jenis-jenis statements dalam JavaScript
  • Menggunakan conditional statements (if, switch) untuk pengambilan keputusan
  • Menerapkan loop statements (while, for, for-of, for-in) untuk perulangan
  • Menggunakan jump statements (break, continue, return) untuk kontrol alur
  • Menerapkan error handling (try-catch-finally) untuk menangani kesalahan
  • Membangun aplikasi sederhana yang mengkombinasikan semua jenis statements

📚 Daftar Script Pembelajaran

1. Expression Statements
Mempelajari expression sebagai statement, compound statements, dan empty statements
2. Conditional Statements
Mempelajari if, if-else, else-if, switch, dan ternary operator untuk pengambilan keputusan
3. Loop Statements
Mempelajari while, do-while, dan for loop untuk perulangan dengan berbagai variasi
4. Advanced Loops
Mempelajari for-of, for-in, array methods, dan teknik iterasi lanjutan
5. Jump Statements
Mempelajari break, continue, return, dan labeled statements untuk kontrol alur program
6. Error Handling
Mempelajari try-catch-finally, throw, custom errors, dan best practices error handling
7. Comprehensive Example
Aplikasi lengkap sistem manajemen mahasiswa yang menggunakan semua jenis statements

📋 Output Console

Pilih script di atas untuk melihat hasilnya... 💡 Tips: • Jalankan script secara berurutan untuk pemahaman yang lebih baik • Perhatikan bagaimana setiap statement bekerja dan berinteraksi • Script 7 (Comprehensive Example) menggabungkan semua konsep yang dipelajari • Gunakan browser Developer Tools (F12) untuk debugging lebih detail
